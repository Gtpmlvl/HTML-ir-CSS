CSS Rašymo būdai

  • Inline - Linijinis rašomas tag'o viduje naudojant style atributą
  • Internal - Vidinis rašomas style tag'o viduje (head dalyje)
  • External - Išorinis rašomas atskirame CSS faile, kurį reikia prijungti prie HTML naudojant link tag'ą

CSS Selector'iai

  • element - Parenkamas elementas pagal jo pavadinimą ( p{} ).
  • id - Sukuriamas susigalvotas ID pavadinimas ir priskiriamas vienam elementui. ( #idPavadinimas{} )
  • class - sukuriamas susigalvotas klasės pavadinimas ir priskiriamas bet kokiam kiekiui elementų ( .klasėsvardas{} ).

CSS Stiliai

  • color - skirtas pakeisti teksto spalvą (color:red;).
  • font-size - skirtas pakeisti teksto dydį (font-size: 20px;)
  • background-color - skirtas pakeisti fono spalvą (background-color:aqua;)
  • font-family - skirtas pakeisti teksto šriftą (font-family: monospace;)
  • font-weight - skirtas pakeisti teksto storumą (font-weight: bold;)
  • line-height - skirtas pakeisti eilutės aukštį (line-height: 20px;)
  • border - skirtas nurodyti rėmelio storį + tipą + spalvą ( border: 5px solid red;)
  • border-radius - skirtas apvalinti rėmelio kampus (border-radius: 10px;)
  • text-align - skirtas lygiuoti tekstą (text-align: center;)
  • font-style - skirtas pakreipti stilių (font-style: italic;)
  • list-styletype - skirtas pakeisti sąrašo žymėjimo tipą.
  • HSL - niekas nenaudoja
  • HEX - naudojamas plačiai (#Ff0000)
  • RGB - naudojamas taip pat plačiai kaip ir HEX. (rgb(255,0,0)

Permatomas spalvos

  • HSL permatomumas nuo 0 (pilnai permatomas) iki 1(pilnai matomas).
