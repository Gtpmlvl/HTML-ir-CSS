Puslapio struktūra

  • header - puslapio viršus, kuriama nurodoma informacija apie puslapį/meniu.
  • nav - skirtas kurti puslapio meniu.
  • footer - pusapio apačia, kurioje yra įvairi informacija (terms and uses, privacy policy, copyrights, links,partneriai).
  • section - didžiulė puslapio dalis.
  • div - maža puslapio dalis.
  • main - pagrindinė puslapio dalis.
  • article - main viduje esanti sekcija.
  • aside - šoninės dalies puslapio sekcija (ne pagrindinė informacija)

Menu info

Kuriant menu - naudojamas nav tag'as, kurio viduje yra sąrašas. To sąrašo elemnetų (li) viduje yra nuorodos tag'as (a), kuris nurodo į kažkurią puslapio dalį arba kitą puslapį.

Nuorodos tag'o (a) atidaromoje dalyje esantis atributas href nurodo į kur vartotojas turi būt nukeltas paspaudus ant nuorodos.

Jeigu atributas nurodo į kažkokį ID ( href= "#ID_pavadinimas"), tuomet vartotojas yra nukeliamas į puslapio dalį kuriai yra suteiktas nurodytas ID (id=ID_pavadinimas).
Puslapio viršus

Jeigu href atributas nurodo į kitą HTML failą (href="./html") , tuomet vartotojas nukeliamas į nurodytą html failą.

Inline, Block ir Inline-Block

Inline

Inline elementai yra tokie elementai, kurie užima tik tiek vietos ekrane, kiek jiems yra reikalinga. Minusas tas, kad negalima keisti šių elemtnų dydžių. Pliusas tas, kad jie lengvai įtalpinami tarp kitų informacijos vienetų.
Tag'ų pavyzdžiai yra : a, span, img, input, button, label,select,textarea.

Block

Block elementai yra tokie, kurie užima visą leistiną plotį ir neįsileidžia kitų elementų į tą pačią eilute. Minusas tas, kad užima visą leistiną plotį ir neįsileidžia kitų. Pliusas, kad galima keisti jų dydį.
Tag'ų pavyzdžiai: h1-h6, p, div, section, main, nav, article, blockqueate, aside, canvas, ol, ul, li etc...
