Naudojant bet kurią apart Statinės pozicijos gauname prieigą prie bet kurių 5 naujų stilių: top,left,right, bottom ir z-index.
Šitie stiliai nurodo per kokį atstumą elemntas turi būti atitrauktas.
Z-index stilius nurodo elemento prioritetą sluoksniuose.( kuris elementas užlips ant kurio elemento esant persidengimui)
