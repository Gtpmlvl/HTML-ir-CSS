CSS Selectors

Visus CSS selektorius galima rasti čia

  • Element - tag'o vardas.
  • .class - tag'ui suteikta klasė.
  • #id - tag'ui suteiktas id.
  • .class1.class2 - tagui suteiktos abi klasės arba daugiau.
  • .class3 .class4 - tag'ui suteikia stilius, jeigu jis turi class4 klasę ir yra class3 palikuonis.
  • * - parenka viską.
  • element1, element2 - i6vardinti ir kableliu atskirti elementus
  • element1 > element2 - element1 vaikinis element2 / element2, kurio tėvinis elementas yra elementas1.
  • a:link - nelankytos nuorodos
  • a:visited - lankytos nuorodos.
  • a:active - aktyvi nuoroda (paspaudimo momentas)
  • a:hover - elementas ant kurio yra užvesta pelė.
